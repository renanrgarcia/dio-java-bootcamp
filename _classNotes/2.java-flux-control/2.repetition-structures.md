# Repetition Structures

## for-loop

```java
for (initialization; condition; update) {
  // code
}
```

### for-loop in arrays

```java
for (int i = 0; i < array.length; i++) {
  // code
}
```

- Abreviated form:

```java
String[] items = {"item1", "item2", "item3"};
for (String item : items) {
  // code
}
```

## while-loop

```java
while (condition) {
  // code
}
```

## do-while-loop

```java
do {
  // code
} while (condition);
```
