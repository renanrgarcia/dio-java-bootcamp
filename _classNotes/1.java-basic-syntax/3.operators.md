# Operators
## Atribution operator
- `=`: assigns a value to a variable
  - Examples:
    `int a = 5;`
    `String name = "John";`
    `boolean isTrue = true;`
    `char letter = 'a';`
    `Date birthDate = new Date();` // class

## Arithmetic operators
- `+`: addition // can be concatenation with strings
- `-`: subtraction
- `*`: multiplication
- `/`: division
- `%`: remainder // module
